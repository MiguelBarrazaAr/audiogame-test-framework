# iku: engine de juegos accesibles

  Iku, motor de creación de videojuegos desarrollado en python 3.7.
Basado en pygame.

##Objetivos:

  Iku es un motor de juegos 2d (game engine) desarrollado para facilitar la creación de audiojuegos o videojuegos accesibles.
Contiene toda las librerías necesarias para el manejo con lectores de pantalla y la manipulación de imagenes en 2d y audio en 2d y 3d.
nota: en face desarrollo aún.

##Requiere las siguientes librerías:
- pygame 1.9.4
- libaudioverse 0.9.1
- accessible-output2
- sound-lib 0.8

##Como instalarlo:

  Se recomienda clonar el repo en tu disco, y luego instalarlo en modo develop con el siguiente comando:

python setup.py develop

de esa forma al  actualizar el repo git con (git pull) tendrás todos los nuevos cambios.
